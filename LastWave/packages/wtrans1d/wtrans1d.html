Description

Package allowing to perform and display 1d (continuous and (bi)orthogonal) wavelet transforms.

Let's run the Demo DemoWtrans1dFrac
Here is the code of the demo:
# Set the current wavelet transform to be a
a
# Generate a devil staircase+gaussian noise
ucantor 0 8000 .334 .5 .332 0 .334 .5
0=prim(0a)+Grand*.01
# Compute the continuous wavelet transform using the first derivative of the Gaussian as the analyzing wavelet
cwtd 1.5 7 10 'g1'
# Display the signal on top of the wavelet transform
disp 0 a
