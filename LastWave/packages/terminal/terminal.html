Description

The terminal package is a script package (no C) that implements high-level interactivity on the command line (the behavior on the command line can be completely controlled using script). This package implements

  • full command line edition
  • history recall (up and down) and history command line completion
  • high level on-line help system. While typing the command line you can access
    • usage or detailed help on the current command
    • command completion
    • usage or detailed help on a current field (i.e., last field name at cursor position)
    • field names completion
    • filenames completion
