Introduction to Lastwave 3.0

 

The examples listed here cover a small part of LastWave functionalities. To learn about them nothing replaces reading the documentation or, at least, running the Demos (just type Demo at startup).

For downloading documentation to any package you should go here
On the left handside there is a list of all the available Lastwave Packages. If you click on one of them you will get
  • a short description of the package
  • the list of authors
  • some output examples (with corresponding scripts) which cover some (not all) aspects of LastWave
